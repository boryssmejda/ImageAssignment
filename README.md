# Image Assignment

## Toolset

The project has been developed on WSL 2 with Ubuntu 22.04.
Following tools were used:
- [CMake](https://cmake.org/), version used 3.27
- [GCC](https://gcc.gnu.org/), version used 12.3.0


## Building the project

```sh
mkdir build
cd build
cmake ../
cmake --build .
```
